Button

Button is one of the most intutive UI element to be used.

  • Straight Edge Buttons

    Straight edge buttons give your design a classy finish and tiled look.

  • Soft Edge Buttons

    Soft edge buttons are slightly edged with very little difference. The main idea is to give the look of Straight edge buttons but with smudged corners.

  • Soft Edge-2x Buttons

    Soft edge-2x buttons expresses a softness in edges which is clear to the eye and gives a calming effect to design.

  • Curved Edge Buttons

    Curved edge buttons are the buttons which we can see with actually curved edges but there is still room for some roundness as these or more on the oval side.

  • Rounded Edge Buttons

    Rounded edge buttons are exactly round and it relives the bit of chaotic edges which were there in the circular edge buttons.

  • Circular Edge Buttons

    Circular edge buttons only have edges in there name. These are forming complete circle.

  • Floating action button

    Class "tui__btn--float-right"/"tui__btn--float-left" can be used to place button at bottom-right or bottom-left.
